แอดมินผู้จัดการ

แผงควบคุมโครงการ Challenge 90 วัน

1. ตั้งค่าวันเริ่มต้นโครงการ

กำหนดวันเริ่มนับ 1 ของโครงการเพื่อเริ่มการติดตามรายสัปดาห์ (ครบ 90 วันอัตโนมัติ)

2. กำหนดที่อยู่ Google Sheets Web App

ระบุ URL ของ Web App หลังจาก Deploy ใน Apps Script เพื่อเปิดใช้งานฐานข้อมูลจริง

3. นำเข้าข้อมูลผลวัดน้ำหนัก/สัดส่วนพนักงานจาก Excel

แอดมินนำเข้าข้อมูลวัดผล **สัปดาห์แรก (เริ่มต้น)** และ **สัปดาห์สุดท้าย (สัปดาห์ที่ 13)** ที่นี่ โดยคัดลอกคอลัมน์ข้อมูลใน Excel แล้วนำมาวางลงในช่องด้านล่าง (คอลัมน์ต้องเรียงลำดับ: รหัสพนักงาน [แท็บ] น้ำหนัก [แท็บ] รอบเอว [แท็บ] รอบสะโพก)

4. บันทึก/แก้ไขสัดส่วนพนักงานรายบุคคล (ด่วน)

ค้นหารหัสพนักงานเพื่อบันทึกหรืออัปเดตข้อมูลน้ำหนักตัว, รอบเอว และรอบสะโพกของสัปดาห์ใดสัปดาห์หนึ่ง (สัปดาห์ที่ 1 - 13) ลงชีตโดยตรงทันที

แผนก/ฝ่าย: -

4. จัดการคำถามแบบประเมินโครงการ

กำหนดหัวข้อคำถามสำหรับใช้คัดกรองพฤติกรรมแรกรับ และคำถามติดตามผลรายสัปดาห์

+ เพิ่มคำถามใหม่

5. สรุปรายงานผลโครงการสำหรับเสนอผู้บริหาร

กราฟภาพรวมโครงการ สัดส่วนเฉลี่ยความสำเร็จรายสัปดาห์ และเปรียบเทียบผล

พนักงานลงทะเบียนทั้งหมด 0 คน
เฉลี่ย % น้ำหนัก/BMI ลดลงสะสม 0% คำนวณจากน้ำหนักแรกเริ่ม
น้ำหนักรวมที่ลดไปแล้ว 0 กก. จากพนักงานที่เข้าร่วมทุกคน

สัดส่วนพนักงานเข้าร่วมแยกตามแผนก (Pie Chart)

ความคืบหน้าภาพรวมรายสัปดาห์ (% การลดลงเฉลี่ยสะสม)

6. ตารางวิเคราะห์ความสัมพันธ์ระหว่างการปฏิบัติพฤติกรรมกับการลด BMI (Correlation Trend)

วิเคราะห์อัตราการลด BMI สำเร็จ (ลดสะสมตั้งแต่ 3% ขึ้นไป) เปรียบเทียบตามความถี่ในการรายงานพฤติกรรมแต่ละด้านของพนักงาน

พฤติกรรมสุขภาพที่บันทึกรายสัปดาห์ อัตราสำเร็จเมื่อปฏิบัติเป็นประจำ (>= 50% ของสัปดาห์) อัตราสำเร็จเมื่อไม่ค่อยปฏิบัติ (< 50% ของสัปดาห์) ผลกระทบของการปฏิบัติ (ต่างกัน)

7. ข้อมูลสถานะและรายชื่อพนักงานทั้งหมด

แสดงผลการติดตามความคืบหน้าส่วนบุคคล คลิกเพื่อดูกราฟของพนักงานแต่ละคนแยกกันได้ทันที

รหัสพนักงาน ชื่อ-นามสกุล แผนก ส่วนสูง น้ำหนักเริ่มต้น ➔ ปัจจุบัน ค่า BMI เริ่มต้น ➔ ปัจจุบัน รายละเอียดรายบุคคล
